# SQL Evaluation – Sakila

Este repositorio contiene la resolución de los ejercicios de SQL utilizando la base de datos **Sakila**.

## Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Importar el archivo `sakila.sql` en el gestor de base de datos MySQL Workbench.
3. Ejecutar las consultas del archivo `consultas sakila evaluacion final.sql` sobre la base de datos Sakila.

El objetivo es practicar consultas SQL como `SELECT`, `JOIN`, `GROUP BY`, `HAVING` y subconsultas.
